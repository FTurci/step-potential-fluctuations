# arrested-demixing

Simulating a shoulder potential of height 1.0 and width 0.5

```
U(r) |....|
     |////|
     |////|
     |////|__.
     |///////|
     |///////|_____ r
     0
```

The folder contains:

- starting configuratons in `startconfs`
- helper modules in `src`
- runs with running script in in `run'
